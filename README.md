# Uncertainty-Quantification-in-PAML


## Description

This repository serves as an initial testing ground for implementing different uncertainty quantification methods in Neural-Netwok-based physical models.

## References

The application in Solid Mechanics and Physics-Aware-Machine-Learning architectures used are based on the course "Tutorial: Machine Learning in Solid Mechanics" at TU-Darmstadt by Prof. Dr. rer. nat Oliver Weeger.
